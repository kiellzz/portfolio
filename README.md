## ⚙️ Running Locally

### 1. Clone the repository

```bash
git clone [https://github.com/kiellzz/portfolio.git](https://github.com/kiellzz/portfolio.git)
cd portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm run dev
```

> 📍 Running at: [http://localhost:3000](http://localhost:3000)
